# OmniFocus Shortcuts for Alfred

Some shortcuts for Omnifocus

## Todo View ( ⌘-T )
Toggle both sidebars.

## Morning Review ( ⇧⌘-M )
Focus on the current Morning Review project at swap to Todo View

## Evening Review ( ⇧⌘-E )
